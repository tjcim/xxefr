Structure of XML

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE name [
  <!ELEMENT name ANY>
]>

<name>Blah</name>
```

* This is the prolog: `<?xml version="1.0" encoding="UTF-8"?>`
* The Document Type Definition (DTD): `<!DOCTYPE name [ <!ELEMENT name ANY> ]>`
* The Document: `<name>Blah</name>`
